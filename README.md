### To build the app
- Clone Repository

```sh
git clone https://github.com/omegaui/android_build_maker
```

- Change Directory
```sh
cd android_build_maker
```

- Build for MacOS
```sh
flutter build macos --release
```

- Look for executable in `build/macos` directory
